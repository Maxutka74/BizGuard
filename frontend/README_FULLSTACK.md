# BizGuard Fullstack

Frontend:
- Оригінальний UI
- Інтеграція API
- OAuth callback
- JWT auth
- Gmail API integration

Backend:
- Django REST API
- Accounts
- Gmail
- Threats
- Dashboard
- Analysis

## Run backend
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

## Run frontend
npm install
npm run dev
